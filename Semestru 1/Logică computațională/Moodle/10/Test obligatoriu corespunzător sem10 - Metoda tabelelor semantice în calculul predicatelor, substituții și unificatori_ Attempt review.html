Asociați regula care se aplică fiecărei formule din coloana din stânga: 

¬ ( (∃x) P(x) ∧ (∃x) Q(x) )

(∃x) P(x) ∧ (∀y) Q(y)

¬ (∃x) P(x)

¬ (∀y) ( P(y) → Q(y) )

Feedback

The correct answer is: ¬ ( (∃x) P(x) ∧ (∃x) Q(x) ) → regula β, (∃x) P(x) ∧ (∀y) Q(y) → regula α, ¬ (∃x) P(x) → regula γ, ¬ (∀y) ( P(y) → Q(y) ) → regula δ
